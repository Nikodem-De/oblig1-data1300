# Hello Health App

## Description: 
This is obligatory assignment that involves properly configuring the development environment. The application verifies Python settings, the repository structure and the GIT workflow. 

## Installation instructions:

1. Clone the repository: `git clone https://github.com/Nikodem-De/oblig1-data1300`

2. Creating a virtual environment: `python -m venv venv`

3. Activating the virtual environment: `.\\venv\\Scripts\\activate`

4. Running the application: `python hello\_health.py`
## Requirements:

* Python 3.10+
* No external dependencies for Oblig 1
